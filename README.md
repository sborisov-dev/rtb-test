# Редактор email-адресов

Тестовое задание от rtb


## Запуск

```
git clone https://bitbucket.org/borisov-sergey/vi.email.editor.git
cd vi.email.editor
npm start
```

Затем перейти по ссылке [http://localhost:8000/](http://localhost:8000/)

## Тесты

unit tests
``npm test``

e2e tests
``npm run protractor``
